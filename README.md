# Bus-Driver-FullStack

This is a full-stack application for managing bus drivers and their tours, built with Django (backend) and React (frontend).

## Live Demo

- **Frontend (React):** [https://bus-driver-full-stack.vercel.app/](https://bus-driver-full-stack.vercel.app/)
- **Backend API (Django):** [https://bus-driver-fullstack.onrender.com/](https://bus-driver-fullstack.onrender.com/)
